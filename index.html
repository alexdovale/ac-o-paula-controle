Termos de Uso do SIGEP

1. Aceitação dos Termos

Ao acessar e utilizar o SIGEP (Sistema de Gerenciamento de Pauta), você concorda integralmente com estes Termos de Uso.

2. Finalidade do Sistema

O SIGEP é uma ferramenta destinada à gestão de atendimentos, visando otimizar o fluxo de trabalho e garantir a organização das pautas.

3. Responsabilidades do Usuário

O usuário se compromete a utilizar o sistema de forma ética, respeitando a privacidade dos dados dos assistidos e seguindo as políticas institucionais.

4. Disponibilidade do Serviço

O SIGEP está disponível 24/7, podendo haver indisponibilidades eventuais por manutenção programada.

Política de Privacidade

1. Natureza do Sistema e Ciclo de Vida dos Dados
O SIGEP coleta e armazena dados pessoais de assistidos (nome, CPF opcional, assunto e demanda) por até 7 dias após a finalização da ação. Após esse prazo, os dados são permanentemente excluídos.

2. Dados Coletados
Para uso do sistema, coletamos apenas dados de autenticação (e-mail e senha) dos profissionais.

3. Proteção de Dados
Seus dados são protegidos com criptografia de ponta a ponta em conformidade com a LGPD.

4. Seus Direitos (LGPD)
Você tem direito a acessar, corrigir ou solicitar a exclusão de seus dados a qualquer momento.
