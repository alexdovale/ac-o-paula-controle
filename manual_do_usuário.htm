1. Introduo

Bem-vindo ao SIGEP, o Sistema de Gerenciamento de Pauta. Esta ferramenta foi desenvolvida para otimizar e organizar o fluxo de atendimentos, permitindo um controle em tempo real de quem est agendado, quem aguarda, quem est sendo atendido e quem j finalizou. Este manual ir gui-lo por todas as funcionalidades do sistema.

2. Primeiros Passos: Acesso ao Sistema

A tela inicial permite que voc acesse sua conta ou crie uma nova.

2.1. Login

1.     Acesse a aba "Login".

2.     Digite seu e-mail e senha cadastrados.

3.     Clique em "Entrar".

2.2. Cadastro de Novo Usurio

1.     Acesse a aba "Cadastrar".

2.     Preencha seu nome completo, e-mail e crie uma senha (mnimo de 6 caracteres).

3.     Clique em "Criar Conta".

4.     Aps o cadastro, sua conta passar por um processo de aprovao pelo administrador do sistema. Voc ser notificado quando o acesso for liberado.

2.3. Recuperao de Senha

1.     Na aba "Login", clique em "Esqueci minha senha".

2.     Digite seu e-mail de cadastro. Um link para redefinio de senha ser enviado para o seu e-mail.

3. Tela Principal: Minhas Pautas

Aps o login, voc ver a tela "Minhas Pautas", que lista todas as pautas que voc criou ou das quais participa.

3.1. Criando uma Nova Pauta

1.     Clique no boto "Criar Nova Pauta".

2.     Tipo de Pauta: Escolha entre "Atendimento Agendado" (para pautas com horrios definidos) ou "Atendimento Avulso" (para atendimentos por ordem de chegada).

3.     Nome da Pauta: D um nome claro para a pauta (ex: "Ao Social Caxias - 10/10/2025").

4.     Ordem de Atendimento:

○      Padro do Sistema (Recomendado): Organiza a fila por um sistema de prioridades baseado na pontualidade.

○      Ordem de Chegada: Organiza a fila estritamente pelo horrio em que a chegada foi registrada.

5.     Fluxo de Atendimento:

○      Sim, usar fluxo de delegao: Habilita a coluna "Em Atendimento", til para equipes onde uma pessoa inicia e outra finaliza o atendimento.

○      No, finalizar diretamente: O assistido vai da coluna "Aguardando" direto para "Atendidos".

6.     Clique em "Criar Pauta".

3.2. Acessando uma Pauta Existente

Clique no card da pauta desejada para abri-la e iniciar o gerenciamento.

3.3. Excluindo uma Pauta

Clique no cone de lixeira no canto superior direito do card da pauta. Uma confirmao ser solicitada antes da excluso permanente.

4. A Interface de Gerenciamento da Pauta

Dentro de uma pauta, voc encontrar a interface principal de gerenciamento, no estilo Kanban.

4.1. Cabealho e Aes Principais

●      Ttulo da Pauta: Exibe o nome da pauta ativa.

●      Boto "Voltar": Retorna tela "Minhas Pautas".

●      Boto "Aes": Abre um menu com funcionalidades avanadas (detalhado na seo 6).

●      Boto "Sair": Desconecta voc do sistema.

4.2. Adicionando Assistidos

No formulrio superior, voc pode adicionar pessoas pauta:

1.     Nome do Assistido: Campo obrigatrio.

2.     CPF: Opcional.

3.     Matria - Assunto: Digite ou selecione o assunto jurdico.

4.     Horrio Agendado: Se aplicvel, defina o horrio.

5.     Assistido j chegou?: Marque "Sim" para envi-lo diretamente para a coluna "Aguardando". Se marcar "No", ele ir para a coluna "Pauta".

6.     Clique em "Adicionar".

4.3. Importando Pauta via Arquivo CSV

Para adicionar mltiplos agendamentos de uma vez:

1.     Clique em "Carregar Pauta (CSV)".

2.     Selecione um arquivo .csv do seu computador.

3.     O arquivo deve ter 4 colunas, separadas por ponto e vrgula ;, na seguinte ordem: Nome Completo;HH:MM;Assunto;CPF(opcional).

4.     Clique no link "Como converter minha pauta?" para ver um exemplo detalhado e um prompt para usar com IAs (como ChatGPT) para formatar pautas a partir de PDFs.

5. Gerenciando o Fluxo de Atendimento (Colunas Kanban)

O fluxo dividido em colunas. Cada assistido representado por um card que pode ser movido entre as colunas.

5.1. Coluna 📋 Pauta

●      O que : Lista os assistidos agendados que ainda no chegaram.

●      Aes:

○      Marcar Chegada: Move o card para a coluna "Aguardando". Voc informar o horrio de chegada.

○      Faltou: Move o card para a coluna "Faltosos".

○      Editar Dados: Permite corrigir informaes do assistido.

○      Excluir: Remove o registro permanentemente.

5.2. Coluna ⏳ Aguardando Atendimento

●      O que : A fila de espera, com assistidos que j chegaram. A ordem definida pela configurao da pauta (Padro ou Ordem de Chegada).

●      Aes:

○      Atender: Move o card para a coluna "Em Atendimento" (se o fluxo de delegao estiver ativo) ou abre o modal para finalizar o atendimento.

○      Prioridade: Move o assistido para o topo da fila (exige um motivo).

○      Editar: Permite editar os dados do assistido.

○      Voltar p/ Pauta: Retorna o assistido para a coluna "Pauta" (disponvel apenas para agendados).

5.3. Coluna 👩‍💻 Em Atendimento

●      O que : (Visvel apenas se o fluxo de delegao estiver ativo) Mostra os assistidos que esto sendo atendidos no momento e por qual colaborador.

●      Aes:

○      Delegar Finalizao: Permite enviar um link por e-mail para que outro colaborador finalize o atendimento.

○      Finalizar Diretamente: Move o card para "Atendidos".

○      Voltar p/ Aguardando: Retorna o assistido para a fila de espera.

5.4. Coluna ✅ Atendidos

●      O que : Histrico de todos os atendimentos finalizados.

●      Aes:

○      Baixar PDF: Gera um relatrio em PDF com todos os atendidos daquela aba (agendado ou avulso).

○      Boto de Confirmao (✔): Permite marcar um atendimento como "verificado" ou "confirmado".

○      Demandas: Adiciona ou edita demandas secundrias para o mesmo atendimento.

○      Editar (Atendente/Dados): Corrige o nome do atendente ou os dados do assistido.

○      Voltar: Reverte o atendimento para a coluna anterior ("Em Atendimento" ou "Aguardando").

5.5. Coluna 🚫 Faltosos

●      O que : Lista de assistidos agendados que no compareceram.

●      Aes:

○      Revertido para Pauta: Move o assistido de volta para a coluna "Pauta" caso ele chegue atrasado.

○      Boto de Confirmao (✔): Permite marcar a falta como "confirmada".

6. Funcionalidades Avanadas (Menu "Aes")

6.1. Editar Nome da Pauta

Permite renomear a pauta atual.

6.2. Compartilhar (Gerenciar Membros)

Convide outros usurios do sistema para visualizar e editar a pauta em tempo real. Basta inserir o e-mail do usurio e clicar em "Convidar". Voc tambm pode remover membros existentes.

6.3. Fechar e Reabrir Pauta

●      Fechar: Bloqueia a pauta para edies. Ningum (exceto o dono) poder fazer alteraes. Requer confirmao por senha.

●      Reabrir: Desbloqueia a pauta, permitindo edies novamente. Requer confirmao por senha.

6.4. Zerar Pauta

Exclui todos os registros de assistidos (de todas as colunas) da pauta atual. Esta ao irreversvel e exige confirmao por senha.

6.5. Colaboradores (Lista de Presena)

Uma ferramenta para gerenciar a equipe de campo.

●      Adicione colaboradores (nome, e-mail (opcional), telefone, forma de deslocamento).

●      Marque a presena e o horrio de chegada de cada um.

●      Envie mensagens via WhatsApp.

●      Gere um PDF com a lista de presena dos que compareceram.

6.6. Anotaes

Abre um bloco de notas privado. As anotaes so salvas localmente no seu navegador e so teis para lembretes pessoais sobre a pauta.

7. Poltica de Privacidade

O SIGAP segue as diretrizes de proteo de dados, garantindo a segurana e a privacidade das informaes.

●      Natureza dos Dados: O sistema uma interface de controle de fluxo. Os dados pessoais dos assistidos (nome, CPF, assunto) so coletados para a gesto do atendimento.

●      Ciclo de Vida: Os dados dos assistidos permanecem no SIGAP por sete dias aps a criao da pauta. Aps esse perodo, so permanentemente excludos. O armazenamento definitivo e oficial dos dados de responsabilidade do sistema Verde da DPERJ, que est em conformidade com a LGPD.

●      Dados Coletados dos Usurios: Para os profissionais que utilizam o sistema, o SIGAP coleta apenas e-mail e senha para fins de autenticao.

●      Origem dos Dados: Os dados dos assistidos podem ser importados manualmente do sistema Verde ou cadastrados diretamente no SIGAP (atendimentos avulsos). O usurio que realiza a importao responsvel pela segurana dos arquivos intermedirios (PDFs, CSVs) em seu dispositivo.

●      Direitos dos Titulares: Os direitos dos assistidos (acesso, correo, etc.) so exercidos diretamente junto ao sistema Verde, que o repositrio final e oficial dos dados.

8. Suporte

Para dvidas, sugestes ou problemas tcnicos, entre em contato com o administrador do sistema.
